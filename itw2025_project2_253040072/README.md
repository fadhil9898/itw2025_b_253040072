## Deskripsi
Project ini merupakan **Project 2** pada mata kuliah **Introduction to Web Technology (ITW) 2025**. Website ini dibuat sebagai latihan pengembangan web statis dengan menerapkan HTML, CSS, JavaScript, serta framework Tailwind CSS untuk menghasilkan tampilan yang responsif dan modern.

## Tujuan
- Memahami struktur proyek web yang terorganisir
- Mengimplementasikan desain responsif menggunakan Tailwind CSS
- Menggunakan JavaScript untuk interaksi dan animasi
- Melatih penggunaan library tambahan seperti Swiper dan ScrollReveal
- Meningkatkan pemahaman UI/UX dasar

## Teknologi & Library
- **HTML5**
- **CSS3**
- **JavaScript**
- **Tailwind CSS**
- **Swiper.js**
- **ScrollReveal.js**
- **Node.js** (untuk dependency & build Tailwind)

## Fitur Website
- Hero section interaktif
- Section about, brands, news, testimonial, dan work
- Slider menggunakan Swiper.js
- Animasi scroll menggunakan ScrollReveal
- Layout responsif untuk desktop dan mobile
- Struktur folder rapi dan terpisah berdasarkan fungsi
